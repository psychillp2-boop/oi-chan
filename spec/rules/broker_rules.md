\# Broker Layer Rules



\## 1. 役割



brokerは外部取引所・MT5などのブローカーと接続し、注文を実行するインターフェース層



\---



\## 2. 責務



\- 注文の送信（buy / sell）

\- ポジション管理の反映

\- 約定結果の受信

\- 接続状態の監視



\---



\## 3. 禁止事項



\- brokerが売買判断を行うこと

\- AI・riskロジックを持つこと

\- marketデータを加工・解釈すること



\---



\## 4. データフロー



execution → broker → 市場



brokerは「実行のみ」を担当する



\---



\## 5. 安全原則



\- 注文は必ずexecution経由のみ受け付ける

\- 不正な注文は即拒否

\- 接続異常時は停止状態に遷移する
